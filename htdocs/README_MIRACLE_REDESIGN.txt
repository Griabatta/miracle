MIRACLE: тестовая новая главная + единая шапка
================================================

Что внутри пакета:

1) index-new.html
   Готовая тестовая страница без PHP. Можно просто загрузить в корень сайта и открыть:
   https://ваш-сайт/index-new.html

2) index-new.php
   Та же главная, но уже с подключением общей шапки и подвала через PHP:
   includes/header.php
   includes/footer.php

3) includes/header.php
   Единая шапка сайта: верхняя полоска, логотип, меню, кнопка Premium BUFO.
   В будущем меню меняется здесь один раз.

4) includes/footer.php
   Единый подвал сайта.

5) assets/css/miracle-redesign.css
   Все стили новой главной.

6) assets/js/miracle-redesign.js
   Мобильное меню + hero-слайдер справа, где меняется только схема камеры.

Как безопасно проверить:

1. Загрузите в корень сайта файлы и папки из архива:
   index-new.html
   index-new.php
   includes/
   assets/

2. Откройте тестовую страницу:
   /index-new.html

3. Если всё нравится, можно заменить главную:
   - старый index.html переименовать в index-old.html
   - index-new.html переименовать в index.html

Про единое меню:

Самый правильный вариант — постепенно перевести страницы на PHP и подключать общую шапку:

<?php
$active = 'home';
include __DIR__ . '/includes/header.php';
?>

Тогда меню будет редактироваться один раз в includes/header.php.

Для старых HTML-страниц единое меню само по себе не подключится, потому что обычный HTML не умеет include. Поэтому есть 2 пути:

1. Постепенно переводить страницы в PHP.
2. Настроить сервер, чтобы .html обрабатывался как PHP. Это удобно, но зависит от хостинга.

Сейчас сделана безопасная тестовая версия, чтобы не ломать действующий сайт.
